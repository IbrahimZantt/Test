# منصة روابط شخصية

صفحة هبوط عربية خفيفة لمشاركة الروابط التعريفية والمهنية مع ثيم فاتح/داكن وبطاقات ديناميكية.

## تشغيل المشروع محلياً
1. ثبّت Python 3 (موجود عادةً مسبقاً على معظم الأنظمة).
2. من مجلد المشروع، شغّل خادم محلي بسيط:
   ```bash
   python3 -m http.server 8080
   ```
3. افتح المتصفح على العنوان `http://localhost:8080` لمعاينة الصفحة.

> بديل سريع: يمكنك أيضاً فتح ملف `index.html` مباشرةً في المتصفح، لكن تشغيل خادم محلي يضمن عمل جميع الأصول بنفس المسارات.

## الاختبارات
لا توجد اختبارات آلية حالياً لأن المشروع ثابت (HTML/CSS/JS فقط). للتمأكد يدويًا:
- تأكد من تحميل الصفحة دون أخطاء في وحدة تحكم المتصفح.
- جرّب تبديل الثيم بين الفاتح والداكن.
- أضف رابطاً جديداً من نموذج «إضافة رابط سريع» وتحقق من ظهوره في قائمة الروابط.
